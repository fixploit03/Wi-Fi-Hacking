## Cara Instal

1. Pake packet manager (`apt`)

   ```
   sudo apt-get update
   sudo apt-get install cowpatty
   ```

2. Compile:
   
   ```
   sudo apt-get update
   sudo apt-get install libssl-dev build-essential git
   git clone https://github.com/joswr1ght/cowpatty.git
   cd cowpatty
   make
   sudo make install
   ```
